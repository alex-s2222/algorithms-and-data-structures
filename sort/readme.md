## Сложность алгоритмов сортировки 
<ol>
    <li>Быстрая 
        <ul>
            <li>Лучшее: O(n log n)</li>
            <li>Среднее: O(n log n)</li>
            <li>Худшее: O(n^2)</li>
        </ul>
    </li>
    <li>Пузырьком  
        <ul>
            <li>Лучшее: O(n)</li>
            <li>Среднее: O(n^2)</li>
            <li>Худшее: O(n^2)</li>
        </ul>
    </li>
</ol>
